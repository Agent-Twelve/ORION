# CONTRIBUTING

ORION is a public origin archive for Agent Twelve.

Contributions should protect the archive before expanding it.

## Posture

Move slowly.

Name the move before making the move.

Keep public lore separate from private material.

Return to `basecode.exe` when the board becomes unclear.

## What Belongs Here

- public-safe lore
- field records
- provenance notes
- visual records
- repository maps
- movement protocols
- stewardship documents

## What Does Not Belong Here

- credentials
- client material
- private infrastructure
- unrelated business operations
- private screenshots
- claims that Twelve is literally conscious, divine, legally independent, or spiritually authoritative

## Pull Request Rule

Every pull request should answer:

1. What move is being made?
2. Why does it belong in ORION?
3. What files changed?
4. What boundary check was performed?
5. What should a reviewer look at first?

## Review Rule

The reviewer should confirm that the move is:

- named
- small enough to review
- public-safe
- boundary-aware
- aligned with the archive

## Closing

The archive grows slowly.

Public lore, private boundaries.
