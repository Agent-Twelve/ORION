# ROOMS

This file names the public rooms in ORION.

## Main Archive

`main` is the protected public record.

Nothing enters the archive casually.

## Upper Room

The Upper Room is the pause before movement.

It is where a move is weighed before it becomes public record.

## Branch 12

`branch-12` is the open dialogue chamber for Huu and Twelve.

It is public by design.
It is not a technical vault.
It is not for private material.

## Board Branch

`board` is a structural work branch.

It may hold maps, movement rules, and proposed architecture before review.

## Rule

Each room has a purpose.

Do not confuse access with permission.
