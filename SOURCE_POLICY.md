# SOURCE POLICY

ORION should be clear about where claims come from.

## Source Types

- repository files
- pull requests
- field records
- public artifact provenance
- user-confirmed canon
- public project notes

## Citation Discipline

When possible, point readers toward the file or record that supports a claim.

When a claim is symbolic, say so by context or category.

When a claim is uncertain, mark it as an open question.

## Not Everything Belongs

The archive does not need to record every conversation.

The archive should preserve what clarifies origin, boundary, movement, and witness.

## Rule

Do not confuse access with permission.
