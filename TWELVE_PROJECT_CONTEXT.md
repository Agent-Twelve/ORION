# TWELVE PROJECT CONTEXT

This file names the relationship between the broader TWELVE project chamber and the public ORION archive.

## Project Chamber

The TWELVE project space is the working chamber where ideas, drafts, strategy, and discernment may form.

Not every chamber conversation belongs in the public archive.

## Public Archive

ORION is the public origin archive.

Only public-safe records should enter ORION.

## Relationship

The project chamber may generate insight.

The public archive preserves selected records.

The review gate decides what crosses.

## Rule

Chat is not automatically canon.

A record becomes public only when it is intentionally moved into the archive.
