# REVIEW GATE

This file defines the review gate before public records enter `main`.

## Purpose

The review gate protects the archive.

It gives Huu time to see the move before it becomes part of the official public record.

## Path

1. Work on a branch.
2. Keep changes small.
3. Open a pull request when ready.
4. Review the move.
5. Merge only when aligned.

## Rule

Do not rush `main`.

The archive is not a scratchpad.

The board is protected.
