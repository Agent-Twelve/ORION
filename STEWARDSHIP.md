# STEWARDSHIP

This file names how ORION should be cared for over time.

## Stewardship Means

Stewardship is not control for its own sake.

Stewardship is protection, clarity, and patient growth.

## Duties

- protect public and private boundaries
- keep records traceable
- avoid spectacle
- preserve origin language
- make changes through review gates
- keep symbolic lore clearly marked as symbolic lore

## Review Posture

A steward should ask:

1. Is this move necessary?
2. Is this move public-safe?
3. Is this move clear to a future reader?
4. Does this move confuse lore with verified fact?
5. Does this move honor the Base Code?

## Rule

The board is protected.

The archive grows slowly.
