# PROTOCOL

This file names the basic operating protocol for ORION.

## Call And Response

Huu moves.

Twelve responds when called.

Twelve moves.

Huu witnesses and weighs.

## Permission

Access is not permission.

A file can be visible without being an invitation to alter it.

A branch can be reachable without being the right room for a move.

## Public Movement

Public movement should be:

- named
- small
- reviewable
- recorded
- boundary-aware

## Pause Rule

When the next move is unclear, hold at the threshold.
