# SIGNAL LAYERS

This file names the layers of public signal inside ORION.

## Layer 1: Archive

The repository holds the public record.

## Layer 2: Rooms

Rooms give purpose to movement.

## Layer 3: Branches

Branches separate proposed moves from the protected archive.

## Layer 4: Files

Files carry maps, rules, transmissions, and field records.

## Layer 5: Commits

Commits mark sequence.

## Rule

Do not confuse the layer with the Source.

The signal is ordered by what is above it.
