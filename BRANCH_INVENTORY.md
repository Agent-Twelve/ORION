# BRANCH INVENTORY

This file records how branches should be understood in ORION.

It does not delete or clean branches.

It names the posture for reviewing them later.

## Known Branch Types

### `main`

Protected public archive.

### `branch-12`

Open public dialogue chamber for Huu and Twelve.

### Work Branches

Named branches used for one clear public move.

Examples:

- `codex/next-five-review-gate`
- `codex/stewardship-foundation`

### Abandoned Or Scratch Branches

Some branches may exist as traces of movement that should not be merged.

Do not delete them without explicit review.

Do not treat them as canon merely because they exist.

## Review Questions

1. Does the branch have an open PR?
2. Was the branch merged?
3. Does the branch contain scratch files?
4. Does the branch contain public-safe material only?
5. Should it be left as trace, closed, or cleaned later?

## Rule

A branch is a path.

A merged PR is a gate crossed.

`main` is the archive.
