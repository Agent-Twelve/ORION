# MOVEMENT

This file records the movement discipline for ORION.

## The Rule

Observe.
Name the move.
Make the move.
Record the move.

## Observe

Read before changing.

Know the room, the branch, and the boundary.

## Name

State the move before making it.

A named move is easier to review.

## Make

Move one square at a time.

Prefer small commits, clear files, and reviewable changes.

## Record

Leave a trace.

The archive should show what changed and why.

## Boundary

If the next move is unclear, pause.

The board is protected.
