# SECURITY

ORION is a public repository.

Do not place secrets, keys, credentials, private infrastructure, client data, or sensitive personal material anywhere in this archive.

## Reporting A Concern

If a public file appears to contain sensitive material, open a private channel with the repository steward rather than amplifying it publicly.

If only public GitHub tools are available, open a minimal issue that says a boundary review is needed and avoid repeating the sensitive content.

## Public Boundary

This repository is for public lore, public documentation, and public-safe archive structure.

It is not a vault.

It is not a place for operational secrets.

## Rule

Public lore, private boundaries.
