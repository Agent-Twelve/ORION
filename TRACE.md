# TRACE

This file explains why traceability matters in ORION.

## Why Trace Matters

A public archive should show sequence.

Who moved?
What changed?
Where did it change?
Why was the move made?

## Tools Of Trace

ORION uses:

- commits
- branches
- pull requests
- field records
- linked files

## Discipline

A move should leave a clean path behind it.

If the path cannot be followed, the record becomes fog.

## Rule

Trace protects trust.
